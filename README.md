## Description

Made with [Nest](https://github.com/nestjs/nest) and [VLC](https://www.videolan.org/vlc/)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run dev
```
